PDF Unlocker — Full User Guide

Batch‑unlock PDFs with multiple passwords, preserve folder structure, copy locked or non‑PDF files, and log everything cleanly.

Overview

This tool processes either a single PDF or an entire folder (including subfolders). It tries multiple passwords from a text file, unlocks PDFs when possible, and mirrors the input folder structure in the output folder. It can optionally copy locked PDFs and non‑PDF files, and write a fresh log for each run.

Unicode‑safe Folder recursion Multiple passwords Config‑driven Deterministic logging

Requirements

  • Python: 3.9+ recommended
  • PyMuPDF: pip install pymupdf
  • Windows paths: Use forward slashes (D:/Folder/File.pdf) or escaped backslashes (D:\\Folder\\File.pdf)
pip install pymupdf

Configuration — settings.json

Place settings.json next to the script. If any of input, output, or password are empty strings, the script will ask you interactively for that location only.

Example: folder mode with full mirroring

{
  "files": true,
  "CopyWithoutPW": true,
  "CopyRestFiles": true,
  "input": "D:/My PDFs/Exam Papers",
  "output": "D:/Unlocked PDFs",
  "password": "D:/password.txt",
  "Log": "Log.txt"
}

Keys

Key Type Meaning Default
files boolean true=folder mode (recursive); false=single file mode false
CopyWithoutPW boolean If a PDF can’t be unlocked, copy the original locked file to output false
CopyRestFiles boolean Copy non‑PDF files to output, preserving structure false
input string Input folder (when files=true) or single PDF path (when files=false) "" → ask
output string Output folder root; subfolders are created automatically "" → ask
password string Path to password.txt (UTF‑8) "" → ask
Log string Log file path/name; if non‑empty, it’s cleared at start and used for this run "" → no log

Paths with spaces: Just wrap the entire path in quotes in JSON, e.g. "input": "D:/My PDFs/Exam Papers". Forward slashes or double backslashes both work.

Password list — password.txt

Provide one candidate password per line. Lines starting with // are treated as comments and ignored. Unicode and special characters are supported.

1234
// common passwords below
secret
my//pass
密碼2025
  • Order matters: passwords are tried top‑to‑bottom; the first match wins.
  • Encoding: save as UTF‑8 to avoid invisible character issues.
  • Comments: only lines starting with // are ignored; inline comments are treated as part of the password.

How it works

  1. Load config: reads settings.json. If input, output, or password are empty, prompts you for each missing one.
  2. Load passwords: reads password.txt, ignoring comment lines.
  3. Discover files:
    • Folder mode: walks the entire input tree; collects PDFs (and other files if CopyRestFiles=true).
    • Single mode: processes the single input PDF.
  4. Process PDFs:
    • Tries each password until one works; saves decrypted PDF to the mirrored output path.
    • If none work and CopyWithoutPW=true, copies the original locked PDF to output; otherwise logs a fail.
    • Already unprotected PDFs are saved directly (no encryption).
  5. Process non‑PDFs: if CopyRestFiles=true, copies them to the mirrored output path.
  6. Logging: if Log is set, the file is cleared at start and all actions are appended during the run.

Result tags

  • [OK] Unlocked or saved (already unprotected)
  • [COPY] Copied original (locked or non‑PDF)
  • [FAIL] Could not unlock and did not copy
  • [ERROR] Unexpected error; may fall back to copy if allowed
  • [SKIP] Output already exists; no overwrite

Mirrored structure

Output paths are built with os.path.relpath to preserve the original folder hierarchy:

Input/
  A/file1.pdf
  B/C/file2.pdf
Output/
  A/file1.pdf
  B/C/file2.pdf

Troubleshooting

Chinese filenames show as %XX

  • This guide’s script uses native filesystem paths and shutil.copy2; it does not URL‑encode. Ensure your paths are plain strings in JSON (quoted), not URLs.
  • Use forward slashes (D:/資料/試題.pdf) or escaped backslashes (D:\\資料\\試題.pdf).

Log says “can’t unlock” but file is unlocked

  • Logs are written once per file after trying all passwords. If you see conflicting lines, confirm you’re using the latest script and that Log.txt is cleared at start.
  • Check password.txt for trailing spaces or invisible characters; save as UTF‑8.

Paths with spaces

  • Just quote the entire path in JSON: "input": "D:/My PDFs/Exam Papers".
  • Do not escape spaces; JSON strings accept spaces directly.

Nothing is logged

  • Set "Log": "Log.txt" (or any filename). If "Log": "", logging is disabled.
  • If the log cannot be cleared at start, the script disables logging and prints an error.

Script (reference)

Save as pdf_unlocker.py. Ensure pymupdf is installed.

import fitz  # PyMuPDF
import os
import json
import shutil
import sys
import tkinter as tk
from tkinter import filedialog

def ask_folder(title):
    root = tk.Tk()
    root.withdraw()
    return filedialog.askdirectory(title=title)

def ask_file(title, filetypes):
    root = tk.Tk()
    root.withdraw()
    return filedialog.askopenfilename(title=title, filetypes=filetypes)

def load_passwords(password_path):
    """Load passwords from file, ignoring lines starting with //."""
    if not password_path or not os.path.exists(password_path):
        print("Password file not found.")
        return []
    try:
        passwords = []
        with open(password_path, "r", encoding="utf-8") as f:
            for line in f:
                line = line.strip()
                if not line or line.startswith("//"):
                    continue
                passwords.append(line)
        return passwords
    except Exception as e:
        print(f"Failed to load {password_path}: {e}")
        return []

def log_message(msg, log_file):
    print(msg)
    if log_file:
        try:
            with open(log_file, "a", encoding="utf-8") as f:
                f.write(msg + "\n")
        except Exception as e:
            print(f"[ERROR] Could not write log: {e}")

def unlock_or_copy_pdf(input_path, output_path, passwords, copy_without_pw, log_file):
    try:
        if os.path.exists(output_path):
            log_message(f"[SKIP] Already exists: {output_path}", log_file)
            return False

        doc = fitz.open(input_path)
        if doc.is_encrypted:
            success = False
            for pw in passwords:
                if doc.authenticate(pw):
                    success = True
                    break
            if success:
                os.makedirs(os.path.dirname(output_path), exist_ok=True)
                doc.save(output_path, encryption=fitz.PDF_ENCRYPT_NONE)
                log_message(f"[OK] Unlocked: {output_path}", log_file)
                return True
            else:
                if copy_without_pw:
                    os.makedirs(os.path.dirname(output_path), exist_ok=True)
                    shutil.copy2(input_path, output_path)
                    log_message(f"[COPY] Could not unlock, copied original: {output_path}", log_file)
                else:
                    log_message(f"[FAIL] Could not unlock: {input_path}", log_file)
                return False
        else:
            os.makedirs(os.path.dirname(output_path), exist_ok=True)
            doc.save(output_path, encryption=fitz.PDF_ENCRYPT_NONE)
            log_message(f"[OK] Already unprotected, saved: {output_path}", log_file)
            return True
    except Exception as e:
        if copy_without_pw:
            try:
                os.makedirs(os.path.dirname(output_path), exist_ok=True)
                shutil.copy2(input_path, output_path)
                log_message(f"[ERROR→COPY] {input_path} → {e}", log_file)
            except Exception as e2:
                log_message(f"[ERROR] Could not copy {input_path} → {e2}", log_file)
        else:
            log_message(f"[ERROR] {input_path} → {e}", log_file)
        return False

def copy_other_file(input_path, output_path, log_file):
    try:
        if os.path.exists(output_path):
            log_message(f"[SKIP] Already exists: {output_path}", log_file)
            return False
        os.makedirs(os.path.dirname(output_path), exist_ok=True)
        shutil.copy2(input_path, output_path)
        log_message(f"[COPY] Non-PDF file copied: {output_path}", log_file)
        return True
    except Exception as e:
        log_message(f"[ERROR] Could not copy {input_path} → {e}", log_file)
        return False

def main():
    # Load settings.json
    try:
        with open("settings.json", "r", encoding="utf-8") as f:
            settings = json.load(f)
    except Exception as e:
        print(f"Failed to load settings.json: {e}")
        sys.exit(1)

    files_mode = bool(settings.get("files", False))
    copy_without_pw = bool(settings.get("CopyWithoutPW", False))
    copy_rest_files = bool(settings.get("CopyRestFiles", False))
    log_file = settings.get("Log", "")
    if not log_file:
        log_file = None
    else:
        try:
            with open(log_file, "w", encoding="utf-8") as f:
                f.write("")
        except Exception as e:
            print(f"[ERROR] Could not clear log file: {e}")
            log_file = None

    # Input path
    input_path = settings.get("input", "")
    if not input_path:
        if files_mode:
            input_path = ask_folder("Choose input folder containing PDFs")
        else:
            input_path = ask_file("Choose input PDF", [("PDF files", "*.pdf")])

    # Output path
    output_path = settings.get("output", "")
    if not output_path:
        output_path = ask_folder("Choose output folder for unlocked PDFs")

    # Password file
    password_path = settings.get("password", "")
    if not password_path:
        password_path = ask_file("Choose password.txt file", [("Text files", "*.txt")])

    passwords = load_passwords(password_path)
    if not passwords:
        print("No passwords loaded.")
        return

    if files_mode:
        if not os.path.isdir(input_path):
            print(f"Input folder not found: {input_path}")
            return
        if not os.path.isdir(output_path):
            os.makedirs(output_path, exist_ok=True)

        all_files = []
        for dirpath, _, filenames in os.walk(input_path):
            for f in filenames:
                all_files.append(os.path.join(dirpath, f))

        if not all_files:
            print("No files found in folder or subfolders.")
            return

        log_message(f"Found {len(all_files)} file(s). Processing...", log_file)
        unlocked, copied_locked, failed, copied_other = 0, 0, 0, 0
        for fpath in all_files:
            rel_path = os.path.relpath(fpath, input_path)
            out_file = os.path.join(output_path, rel_path)
            if fpath.lower().endswith(".pdf"):
                result = unlock_or_copy_pdf(fpath, out_file, passwords, copy_without_pw, log_file)
                if result:
                    unlocked += 1
                else:
                    if copy_without_pw:
                        copied_locked += 1
                    else:
                        failed += 1
            else:
                if copy_rest_files:
                    if copy_other_file(fpath, out_file, log_file):
                        copied_other += 1

        log_message(
            f"\nDone. {unlocked} unlocked, {copied_locked} copied (locked PDFs), {failed} failed, {copied_other} other files copied.",
            log_file
        )

    else:
        if not os.path.isfile(input_path):
            print(f"Input file not found: {input_path}")
            return
        if not os.path.isdir(output_path):
            os.makedirs(output_path, exist_ok=True)

        out_file = os.path.join(output_path, os.path.basename(input_path))
        unlock_or_copy_pdf(input_path, out_file, passwords, copy_without_pw, log_file)
        log_message("\nDone.", log_file)

if __name__ == "__main__":
    main()

Best practices

  • Keep logs clean: set "Log": "Log.txt" to get a fresh log each run.
  • Use UTF‑8: save password.txt in UTF‑8 for reliable matching.
  • Avoid overwrites: the script skips existing outputs; delete stale files if you want a fresh run.
  • Structure: choose an output root that’s separate from input to avoid confusion.
